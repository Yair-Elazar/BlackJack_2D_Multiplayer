# BlackJack_2D_Multiplayer

## Project Overview

**BlackJack_2D_Multiplayer** is a Unity project that offers a modern take on the classic game of Blackjack, focusing on clean architecture, modular core game logic, and scalability for multiplayer gameplay. The architecture ensures that core gameplay is isolated, making the system robust and maintainable. The project is structured for both future enhancements and portfolio presentation.

## Tech Stack

- **Unity** (2D, latest version recommended)
- **C#**
- **.NET** (Managed by Unity)
- **Clean Architecture** principles
- Prepared for **multiplayer** extensions (Photon, Mirror, or Unity's Netcode-ready)

## How to Run

1. **Clone the repository:**
    ```
    git clone https://github.com/your-username/BlackJack_2D_Multiplayer.git
    ```
2. **Open in Unity:**
    - Launch Unity Hub
    - Click `Add` and select the project folder
    - Open the project in Unity (2D mode)
3. **Build and Play:**
    - Open the main scene (to be created)
    - Click the Play button in the Unity Editor

> **Note:** This project follows Unity's recommended folder structure and uses `.gitignore` for optimal version control.
